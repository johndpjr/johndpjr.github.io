Learning Experience: PeachOS

PeachOS is a 32-bit operating system (OS) written in C that I developed alongside a Udemy course that taught building a multi-tasking OS from scratch. Having written over 3.5k lines of code and watching 27 hours of video content, I learned so much about systems programming and managing software complexity.

Personal Project: Clocker

Clocker is a Command Line Interface (CLI) written in Python that tracks your work hours. During my last internship, I needed a good way to track my hours for the day, so I created this application to track clocking in and out of work and lunch. I love applying software solutions to real needs!
